欢迎来到万圣节恶魔的领地！

9/16/2025

我常用的网名是 Cytrogen，“万圣节恶魔” 是我懒得改掉的名字。

2004 年冬天，我出生在山东省的一个三线小城市。2015 年夏天，我随着家里人搬到了美国纽约。2023 年秋天，我进入了纽约市立大学伯鲁克分校学习计算机科学专业。

编程历程

2020 年

2020 年疫情爆发，无所事事的我突然有了新的想法：学习编程。那会儿对 Galgame 十分感兴趣，想要自己制作一个，不过我最初接触的并非编程语言，而是 OnScripter 引擎的脚本语言。当时网上的 OnScripter 教程多数都没有做完，我做出来的东西也不怎么样，很快我便放弃继续使用 OnScripter。

但还没完，我对游戏开发的兴趣并没有消失。我学习了 C++ 的基础语法，想要自己做一个控制台内的文字冒险游戏。然而，我并没有学 C++ 多深，以至于代码写得很糟糕，我也放弃了这个项目。

2021 年

2021 年我接触到了 QQ Bot。借此我学习了 Python，为接下来三年的 “人生苦难，我用 Python” 埋下了伏笔。基于 Nonebot2 和 go-cqhttp，我开发了 HokaBot。现在这个项目因为各种原因已经停止维护、无法使用。 HokaBot 的开发期间，我用 Hexo + Netlify 搭建并部署了这个博客网站。如果你有往最前面看的话，可以看到我的第一篇文章是 2022 年 2 月份发布的 Nonebot beta1 教程。

2022 年

2022 年我开始更注重于学业，暂停了多数的编程活动，除了 2 月份中旬和朋友们参加了一次 GameJam，我们的作品是一个基于 Unity 的 2D 平台游戏，但是因为各种原因没有按时完成，做出的实品也和预期中的差了十万八千里。这件事让我感到自尊受损，同年暑假再度参加 GameJam，制作了一个种田游戏。这个游戏因为个人原因没能按照预期完成，也让我更清楚地认识了自己的能力，以及我是否真的热爱开发游戏。

2023 年

2023 年我系统性地学习了网页爬虫和 JS 逆向、陆续完成了哈佛大学的三个 CS50 课程，并开发了 Hexo 主题Anatole-Core-Resume，该项目是我为了搭建一个在线简历网站而制作的，因看不懂 Hexo 的文档、该项目十分简陋和粗糙，亦有改进的地方。 年底我计划在大二完成 IBM 的全栈云应用开发微学士学位，目前每完成一个证书我都会在这个博客上发布一篇文章。

2024 年

2024 年年初我受同校的学长邀请，加入了一个创业项目。这个项目还在构思阶段，我负责的是前端开发，技术栈最先是 React + TailwindCSS、后端用的是 Flask。初次合作的经历令人难以形容，省流来说、算是提前遇到了一支糟糕的管理 + 不合理的项目规划的团队吧。 目前这个项目被打回重做，未来的技术栈可能会变成面向移动端的 React Native。 现在该项目已被确认暴死。

因为 IBM 的全栈云应用开发微学士学位的课程内容和我目前所在团队的技术栈有很大的重合，我便决定在项目还在构思的阶段期间，先自行学习实践全栈项目。于是乎，Hotaru.TS 诞生了。这是一个原先基于 React + Express + MongoDB 的项目，之后因为我想要使用 TypeScript，所以后端改用了 NestJS。目前这个项目 还在开发中 已经暂停开发，你可以在这个博客网站中找到我对这个项目的一些记录。

七月，我如愿结束了 13 个课程、斩下 13 个证书，获得了 IBM 全栈应用开发微学士（期间 edX 添加了一个 Git 和 GitHub 相关的课程，并将微学士课程改名为全栈应用开发）。由于想要尽快积攒实战经验的心态，我开始使用 Forage 来做实习模拟。 很有可能的是，我会放弃继续学习 NestJS 框架。虽然说是有可能，但近日我一定会大幅度减缓 NestJS 学习文章的发布速度。

NestJS 真好玩。八月份我因为简历，开始开发一个叫 ResumeRocket 的简历网站，同时在开发期间学习使用 Atlassian 旗下的各个工具，也是逐渐让自己从使用 GitHub、转到使用 BitBucket。不过这个个人项目在后续的各方面考虑下，停止了。

九月份我开始研究大学的网站，并从中发现了许多可以用浏览器插件来优化的机会：简单的有自动登录，难点的有选课帮手。研究过程中，我意外在某页面上发现了一个逆天数据包：字符长达 60 万的 JSON 文件，里面充斥着大量令人匪夷所思的条件逻辑，结构跟 if-else 嵌套没什么区别。本来不会很困难的功能，因为这个数据包，我先是在扩展程序内部直接处理这些数据，之后又搭建了个代理服务器、处理完数据后再将精简的数据传到扩展程序。那时候我还将这个项目命名为：BetterBaruch。现在这个项目因为工作原因，处于半死不活的状态，顶多开学季会折腾一下。

十一月份，一个学期都要结束了，我意识到一个简历上项目空空是不好的，至少也得是一个完整的、能跑的项目。因此，我开始开发 Shopping-Nest，一个 React + NestJS 的购物平台网站。现在关于它的博客内容还在更新中。（哈哈因为工作也鸽掉了）

2025 年

三月，我开始工作…… 实际上工作从去年十二月就开始了。这期间无聊的时候，都会弄一些简单的工具网页出来。其中就有我对 React Native 的初次尝试。大学的官网网页永远都是一年一新、旧的放着不维护…… 这包括了社团网页。借此，我开发了 BetterBaruchRN。其实这是一个很简单的客户端 APP，早期它没有服务端，数据全靠爬（现在去这个仓库里安装发行版应该也是纯客户端的版本，非常简陋）。后来我想要搭建它的服务端、扩展功能，不过因为工作停止了。

个人爱好

我在童年时期开始就喜欢画画和创作，不过因为种种原因，我并没有继续这个爱好。归根结底是技术上的不足、锻炼少以及只想画自己想画的东西。

音乐喜欢听ボカロ，最喜欢的 P 主是 john，代表作是春嵐。讽刺的是，我并不会唱歌，也不会弹奏任何乐器 最近在学习如何随着五线谱弹奏钢琴。

游戏方面，我喜欢玩沙盒、休闲、模拟经营游戏。我经常会在游戏厅里玩舞萌 DX， 现在 W3。

我对硬件有兴趣。我喜欢组装电脑，也喜欢跟着网上的视频组装一些小东西比如蓝牙音响，但是涉及的范围不广、反而非常狭隘。如果我有机会的话，完成天气预报显示屏会是我最想做的事情。
