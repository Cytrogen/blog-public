Font Requirements & Subsetting Instructions

Required Font Files (WOFF2 format only)

1. Open Sans

Download from: Google Fonts (https://fonts.google.com/specimen/Open+Sans)
Required weights: Regular (400), Bold (700)
Subset: Latin (U+0000-00FF, U+0131, U+0152-0153, U+02BB-02BC, U+02C6, U+02DA, U+02DC, U+2000-206F, U+2074, U+20AC, U+2122, U+2191, U+2193, U+2212, U+2215, U+FEFF, U+FFFD)

Files to create:

  • opensans-regular-latin.woff2
  • opensans-bold-latin.woff2

2. Noto Sans SC (Simplified Chinese)

Download from: Google Fonts (https://fonts.google.com/specimen/Noto+Sans+SC)
Required weights: Regular (400), Bold (700)
Subset: Chinese Simplified (U+4E00-9FFF, U+3400-4DBF, U+20000-2A6DF, U+2A700-2B73F, U+2B740-2B81F, U+2B820-2CEAF, U+F900-FAFF, U+2F800-2FA1F)

Files to create:

  • notosanssc-regular-chinese.woff2
  • notosanssc-bold-chinese.woff2

3. JetBrains Mono

Download from: JetBrains Official (https://github.com/JetBrains/JetBrainsMono/releases)
Required weights: Regular (400), Bold (700)
Subset: Latin Extended + Programming symbols (U+0000-00FF, U+0131, U+0152-0153, U+02BB-02BC, U+02C6, U+02DA, U+02DC, U+2000-206F, U+2074, U+20AC, U+2122, U+2191, U+2193, U+2212, U+2215, U+FEFF, U+FFFD, U+0100-024F, U+1E00-1EFF, U+2020, U+20A0-20AB, U+20AD-20CF, U+2113, U+2C60-2C7F, U+A720-A7FF)

Files to create:

  • jetbrainsmono-regular-latin.woff2
  • jetbrainsmono-bold-latin.woff2

Subsetting Tools

  • Google Fonts: Use the Google Fonts API with appropriate subset parameters
  • Font Squirrel: https://www.fontsquirrel.com/tools/webfont-generator (Expert mode with custom subsetting)
  • Fonttools: pyftsubset font.ttf --unicodes="U+0000-00FF,U+0131,..." --flavor=woff2
