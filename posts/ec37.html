Linux Desmume安装

8/1/2022

该文档讲述了如何在 Linux 系统下安装最新版本的 Desmume。

开始阅读前，需要知道：

  • Desmume 是一个 NDS 模拟器
  • NDS 是任天堂在 04 年发售的便携式游戏机，具有双屏幕显示、触摸屏等功能或特色
  • 该文档搬运且汉化了 Wiki 上的安装方法：用 Git 安装
————————

获取源码

如果没有Git

  1. 进入 Desmume 的 GitHub 仓库。
  2. 点击绿色的 Code 按钮，点击 Download ZIP。
  3. 解压后备用。

反之有Git

  1. 打开终端，输入git clone https://github.com/TASVideos/desmume
————————

安装

  1. 打开终端，输入以下：

    如果想使用 offscreen mesa 作为替补 3D 引擎，改用 ./configure --enable-osmesa。

    如果想启用 HUD 显示，则用 ./configure --enable-hud。

  2. 安装成功，在终端中输入 desmume 或在应用中点击其图标即可开启。
